📋 Panduan Konfigurasi Lengkap
1
Buat akun Cloudflare
Daftar gratis di cloudflare.com. Plan gratis sudah cukup — R2 free tier: 10GB storage, 10 juta operasi/bulan.
2
Buat R2 Bucket
Dashboard → R2 Object Storage → Create Bucket → Nama: aetherislog-photos → Region: Auto → Create.
Lalu masuk ke bucket → Settings → Public Access → Enable R2.dev subdomain.
3
Buat Cloudflare Worker
Workers & Pages → Create → Worker → Name: aetherislog-upload → Deploy.
Edit kode Worker dengan script di bawah ini:
// aetherislog-upload Worker export default { async fetch(request, env) { const cors = { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'PUT,DELETE,OPTIONS', 'Access-Control-Allow-Headers': 'Content-Type,X-Auth-Token,X-Item-Id', }; if (request.method === 'OPTIONS') return new Response(null, { headers: cors }); const token = request.headers.get('X-Auth-Token'); if (token !== env.UPLOAD_SECRET) return new Response('Unauthorized', { status: 401, headers: cors }); if (request.method === 'PUT') { const id = request.headers.get('X-Item-Id') || 'unknown'; const ext = request.headers.get('Content-Type')?.includes('png') ? 'png' : 'jpg'; const key = `habits/${id}/${Date.now()}.${ext}`; await env.HABIT_BUCKET.put(key, request.body, { httpMetadata: { contentType: request.headers.get('Content-Type') } }); const url = `https://${env.R2_PUBLIC_DOMAIN}/${key}`; return Response.json({ success:true, url, key }, { headers: cors }); } if (request.method === 'DELETE') { const { key } = await request.json(); await env.HABIT_BUCKET.delete(key); return Response.json({ success:true }, { headers: cors }); } return new Response('Not found', { status:404, headers:cors }); } }
4
Tambahkan R2 Binding di Worker
Worker → Settings → Bindings → Add → R2 Bucket:
• Variable name: HABIT_BUCKET
• R2 bucket: aetherislog-photos
5
Tambahkan Environment Variables
Worker → Settings → Variables → Add:
• UPLOAD_SECRET = token rahasia Anda (buat sendiri, misal: alog-s3cr3t-2026)
• R2_PUBLIC_DOMAIN = domain publik bucket (dari langkah 2)
6
Konfigurasi CORS pada R2 Bucket
Bucket → Settings → CORS → Add Rule:
[{ "AllowedOrigins": ["*"], "AllowedMethods": ["GET","PUT","DELETE"], "AllowedHeaders": ["*"], "MaxAgeSeconds": 3600 }]
7
Isi form konfigurasi di atas & test koneksi
Setelah semua langkah selesai, isi Worker URL, Secret Token, dan R2 Domain di form konfigurasi di atas, lalu klik "Test Koneksi". Jika berhasil, foto yang kamu unggah akan otomatis tersimpan di Cloudflare R2!

🛠️ Panduan Setup Per Provider
🟢 SUPABASE STORAGE — Rekomendasi Utama (Gratis 1GB)
1
Aktifkan Storage di Supabase Dashboard
Buka supabase.com → project kamu → menu Storage → klik "New bucket"
Nama: habit-photos · centang Public bucket → Create
2
Set RLS Policy (izin upload)
Storage → habit-photos → tab Policies → New Policy → pilih template "Allow authenticated users to upload"
-- Atau jalankan di SQL Editor: CREATE POLICY "Users upload own photos" ON storage.objects FOR INSERT TO authenticated WITH CHECK (bucket_id = 'habit-photos' AND (storage.foldername(name))[1] = auth.uid()::text);
3
Set di index.html dan deploy
const STORAGE_PROVIDER = 'supabase'; // ← sudah benar const SUPABASE_BUCKET = 'habit-photos'; // ← sudah benar // SUPABASE_URL dan SUPABASE_ANON sudah diisi → selesai! ✅
🔵 CLOUDINARY — 25GB Gratis, Tanpa Backend
📁 Struktur Folder Otomatis di Cloudinary:
bayu/lari-pagi/1716123456-ab3f.jpg bayu/meditasi/1716234567-cd5e.jpg sari/olahraga/1716345678-ef6g.jpg demo-user/habit/1716456789-gh7i.jpg
Semua foto semua user terkumpul di 1 akun Cloudinary milikmu. Filter di Media Library dengan tag aetherislog atau user_bayu.
1
Daftar & ambil Cloud Name
cloudinary.com → daftar gratis → Dashboard → salin Cloud name
2
Buat Unsigned Upload Preset
Settings → Upload → Upload presets → Add upload preset
Signing mode: Unsigned · Preset name: aetherislog_unsigned
Folder di preset: kosongkan (diatur otomatis oleh kode)
3
Set di index.html — hanya 2 baris
const STORAGE_PROVIDER = 'cloudinary'; const CLOUDINARY_CLOUD_NAME = 'namamu'; // dari dashboard const CLOUDINARY_UPLOAD_PRESET = 'aetherislog_unsigned'; // Foto otomatis masuk ke folder: {username}/{nama-habit}/
4
Monitor semua user di Cloudinary Media Library
cloudinary.com/console/media_library
Filter by tag: aetherislog → semua foto app ini
Filter by tag: user_bayu → foto milik user bayu saja
🟣 IMAGEKIT — 20GB Gratis + CDN Transformasi
1
Daftar & ambil kredensial
imagekit.io → daftar gratis → Developer Options → salin Public Key dan URL Endpoint
2
Set di index.html
const STORAGE_PROVIDER = 'imagekit'; const IMAGEKIT_PUBLIC_KEY = 'public_xxx'; const IMAGEKIT_URL_ENDPOINT = 'https://ik.imagekit.io/your_id'; // IMAGEKIT_AUTH_ENDPOINT: opsional, untuk private upload
💡 Rekomendasi Developer
Mulai dengan Supabase Storage karena sudah terintegrasi penuh (auth + database + storage satu tempat).
Jika butuh lebih dari 1GB, upgrade ke Cloudinary — upload langsung dari browser tanpa backend, 25GB gratis, CDN global.
